# IBM Credit Card Transaction Analysis

## 프로젝트 개요
IBM 합성 신용카드 거래 데이터(2,400만 건)를 활용한 
핀테크 유저 행동 분석 포트폴리오

## 분석 목표
- AARRR 기반 유저 퍼널 분석
- Cohort 분석 및 LTV 산출
- 데이터 기반 가설 설정 및 A/B 테스트 설계·검증

## 기술 스택
- Python (pandas, scipy, statsmodels)
- SQL (SQLite / MySQL)
- Tableau

## 데이터
- IBM Synthetic Credit Card Transactions (Kaggle)
- 2,000명 유저 / 24,386,900 거래 / 6,146개 카드
